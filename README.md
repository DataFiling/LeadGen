#Real Estate Lead API
**Extract High-Potentiality Leads from Messy Public Records.**

This API uses advanced parsing to identify distressed properties 
before they hit the mainstream market. 

### Endpoints
- `GET /leads/{zip_code}`: Returns a list of properties in the specified area.

### Authentication
Secure your requests using your RapidAPI Key. The 'Watchers' are always 
monitoring for unauthorized access.
